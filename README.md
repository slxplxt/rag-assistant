# RAG Assistant 

AI-ассистент для ответов на вопросы по банковским тарифам с удобным **веб-интерфейсом**. Проект использует архитектуру **RAG (Retrieval-Augmented Generation)** для поиска точной информации в документах и генерации ответов с помощью нейросети, полностью исключая придумывание фактов (галлюцинации).

## 🚀 Возможности
* **Веб-интерфейс:** Красивый и интуитивно понятный чат в браузере, написанный на `Streamlit`.
* **Извлечение текста:** Парсинг сложных банковских PDF-документов с помощью `PyMuPDF`.
* **Умный чанкинг:** Разбивка текста на смысловые фрагменты через `LangChain`.
* **Векторный поиск:** Использование локальной базы данных `ChromaDB` и легковесных эмбеддингов `rubert-tiny2` для быстрого семантического поиска.
* **Генерация ответов:** Интеграция с API **GigaChat** для формирования вежливых, аргументированных и точных ответов на основе найденного контекста.

## 🛠 Стек технологий
* **Python 3.12**
* **Streamlit** (веб-фреймворк)
* **LangChain** (фреймворк для LLM)
* **ChromaDB** (векторная база данных)
* **HuggingFace / sentence-transformers** (модели эмбеддингов)
* **GigaChat API** (языковая модель)
