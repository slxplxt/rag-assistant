#RAG Assistant

Консольный AI-ассистент для ответов на вопросы по банковским тарифам. Проект использует архитектуру **RAG (Retrieval-Augmented Generation)** для поиска точной информации в документах и генерации ответов с помощью нейросети, минимизируя галлюцинации.

## 🚀 Возможности
* **Извлечение текста:** Парсинг сложных банковских PDF-документов с помощью `PyMuPDF`.
* **Умный чанкинг:** Разбивка текста на смысловые фрагменты с учетом токенов через `LangChain`.
* **Векторный поиск:** Использование локальной базы данных `ChromaDB` и эмбеддингов `rubert-tiny2` для семантического поиска.
* **Генерация ответов:** Интеграция с API **GigaChat** для формирования вежливых и точных ответов на основе найденного контекста.
* **Интерактивный чат:** Удобный консольный интерфейс для общения с ботом в реальном времени.

## 🛠 Стек технологий
* **Python 3.12**
* **LangChain** (фреймворк для LLM)
* **ChromaDB** (векторная база данных)
* **HuggingFace / sentence-transformers** (модель эмбеддингов)
* **GigaChat API** (языковая модель)
* **PyMuPDF** (работа с PDF)
